# Security Policy

## Supported Versions

This project is maintained on a best-effort basis. Security fixes will be applied to the default branch (`main`).

## Reporting a Vulnerability

If you discover a security vulnerability, please do **not** open a public issue.

Instead, email:

- Oussama Louati — `oussamalouatioussama@gmail.com`

Please include:

- A clear description of the issue
- Steps to reproduce / proof of concept
- Potential impact

We will acknowledge receipt and work on a fix as soon as possible.
